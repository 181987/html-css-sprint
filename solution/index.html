Content first

Get familiar with the most useful HTML tags and write a simple page using most common tags like headers, paragraphs, images and lists. Don't forget the HTML skeleton and important metatags in the head section.

CSS for colors & fonts

Write your first CSS code to design the fonts and colors of your page. Also add some basic style for links and images. Pick your fonts from Google fonts and use a color picker to copy colors from a website you like.

Improve your page's structure & design

It's time to improve your page's structure by adding some div and by designing all your page's sections using class and id selectors. Class and id enable you to customize the design of specific elements in your page. Use class as much as you can and use an id for unique elements.
